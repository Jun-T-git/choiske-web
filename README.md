# チョイスケ Web サービス

「チョイスケ」は、会員登録不要・無料で使えるシンプルなスケジュール調整 Web サービスです。

---

## プロジェクト概要

- 直感的な UI で日程調整が可能
- 回答集計・決定・通知までワンストップ
- 広告モデルによる無料提供
- Next.js + TypeScript ベース
